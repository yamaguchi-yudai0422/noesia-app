What is Noesia?
Noesia is a persona reproduction AI.
It learns how you actually think — your contradictions, hesitations, and emotional patterns — and reproduces them as a "persona" you can talk to.
Unlike personality tests or coaching tools, Noesia doesn't try to make your thinking more coherent. It preserves the mess.
🔗 noesia.jp | noesia.onrender.com

Key Features

Formation Mode — Answer questions daily. A persona grows from your real responses.
Reproduction Mode — Talk to an AI constrained by your persona's actual tendencies. No smoothing, no meta-analysis.
Comment Mode — Get a third-person analysis of any persona's patterns.
Multiple Persona Slots — Each slot grows independently. Personas never "complete."

Getting Started

Open noesia.onrender.com — no account needed to start
Start Formation Mode and answer questions. There are no right answers, just be honest.
Keep answering to level up your persona
Try Comment Mode to see a third-person analysis of your persona
Then try Reproduction Mode to talk directly to your persona

Noesiaとは？
Noesia（ノエシア）は人格再現AIです。
あなたの思考傾向・感情の揺れ・判断の癖を学習し、それを「人格」として再現・対話できるAIシステムです。
性格診断でもコーチングツールでもありません。矛盾や迷いをそのまま保存・再現することを目的としています。
🔗 noesia.jp | noesia.onrender.com

主な機能

形成モード — 質問に答えることで人格データが蓄積されます
再現モード — あなたの人格に縛られたAIと対話できます。客観視・中庸化はしません
人格コメントモード — 第三者視点で人格の傾向を分析します
複数スロット — スロットごとに独立した人格が育ちます。人格は「完成」しません
